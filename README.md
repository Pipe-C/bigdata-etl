# Proyecto Big Data - ETL

## Descripción
Proyecto académico de la asignatura Fundamentos de Big Data.
Incluye procesos ETL para carga de departamentos, municipios y operaciones.

## Contenido
- Scripts SQL de creación de tablas, vistas y consultas
- Scripts Python para procesos ETL
- Correcciones para algunas tablas, adiciones de campo y formatos incorrectos de fecha.
- Consultas analíticas para gráficos de Pareto y de Torta.

## Tecnologías
- PostgreSQL
- Python
- Git / GitHub
